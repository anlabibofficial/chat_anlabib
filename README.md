🕵️‍♂️ Chat_anlabib
A lightweight, blazing-fast, and completely untracked real-time chat application. No logs, no history, no traces.

Welcome to Chat_anlabib! This project provides a simple, secure space for real-time conversations. Whether you want to quickly sync up with friends or host a temporary discussion, this app lets you drop in, chat anonymously, and vanish without leaving a footprint.

✨ Features:
Zero Tracking & No History: Chats are strictly live. Nothing is saved to a database, and new room members cannot read messages sent before they joined.

One-Step Room Creation: The UI is incredibly simple. Type a room name in the home screen—if it exists, you join it. If it doesn't, it's created instantly.

Total Anonymity: Pick any username you want to mask your identity. No accounts, no passwords, no emails required.

Auto-Wipe on Leave: The moment you leave a room, the entire chat interface (both your messages and others') vanishes from your screen completely.

Real-Time Communication: Powered by WebSockets for instant, lag-free messaging.

100% Free: No premium tiers or locked features.

🛠️ Tech Stack
Frontend:

- HTML5

- CSS3

- Vanilla JavaScript

Backend:

- Node.js

- Express.js

- Socket.io (for real-time, bi-directional communication)

🚀 How It Works
Open the App: You will be greeted by a minimalist home screen.

Enter Details: Fill in two fields:

Room Name: Type a unique name to create a new room, or type an existing name to join an active room.

Username: Enter your anonymous alias.

Start Chatting: Send messages instantly.

Vanish: Close the tab or hit the "Leave" button to wipe the session from your device entirely.

💻 Local Installation
Want to run this project on your local machine or contribute to its development? Follow these steps:

1. Clone the repository:

Bash
git clone https://github.com/anlabibofficial/chat_anlabib.git

cd chat_anlabib

2. Install dependencies:
Make sure you have Node.js installed, then run:

Bash

npm install

3. Start the server:

Bash

npm start

# or if you are using node directly:

node server.js

4. Open in browser:

Navigate to http://localhost:3000 (or whichever port your app uses) to see the app in action!
